# Symbol types
Symbol type 	Description
A 				Global absolute symbol
a 				Local absolute symbol
B 				Global bss symbol
b 				Local bss symbol
D 				Global data symbol
d 				Local data symbol
f 				Source file name symbol
R 				Global read-only symbol
r 				Local read-only symbol
T 				Global text symbol
t 				Local text symbol
U 				Undefined symbol
